Configuring TextPad for Tea

 

The following steps to help you get started with Tea in TextPad:

1.         Enable Tea syntax highlighting

2.         Add the Tea compiler as a tool in TextPad

3.        Add a filter for Tea files to the Open dialog box

1. Enable Tea syntax highlighting

Before you can do syntax highlighting with Tea files, you will need to obtain its syntax definition file, tea.syn.  This can be downloaded from http://se-mmasse/teac/tea.syn.  Copy this file to your TextPad installation’s System directory.  Next follow these steps:

1.         From the Configure menu, choose New Document Class. The Document Class Wizard is displayed.

2.         In the Documet Class Name box, type Tea and click Next.

3.         In the Class Members box, type *.tea and click Next.

4.         Check the box to enable syntax highlighting.

5.         From the drop-down list, choose tea.syn and click Next.

6.         Click Finish.

2. Add the Tea compiler as a tool in TextPad

1.         From the Configure menu, choose Preferences. The Preferences dialog box is displayed.

2.         Select "Tools".

3.         Click Add, and choose the option "DOS Command".

4.         Type in a command of the form:  java -cp Toolbox.jar org.teatrove.toolbox.Teac <options>.  This is the command that will run the Tea compiler on an individual file or set of files.  The exact syntax and options of Teac can be found here.  The TextPad special variable documentation is available here.  Choose the command that works best for your needs. 

TIP:  Consider placing TextPad workspace files at the root directory of your templates and using the $WspDir variable as a parameter to the Teac sourcepath option.

5.         Optionally rename the tool by clicking it twice, slower than double click speed, and typing Teac;. You may insert an "&" before the character you want to use as a menu mnemonic.

6.         Click Apply and then Select "Teac".

7.         Type the following text in the Regular expression text field:  \(^.*\.tea\):\([0-9]+\):

8.         Select 1 for the "File" Register and 2 for the "Line" Register.  This will allow you to jump to errors in your Tea files.

9.         If necessary, click the up arrow to move the new tool to the required position on the menu.

10.     Click OK.

 

3. Add a filter for Tea files to the Open dialog box

1.         From the Configure menu, choose Preferences. The Preferences dialog box is displayed.

2.         Select "File Name Filters".

3.         Click New.

4.         Change the description to something like Tea or Tea (*.tea).

5.         Type *.tea, in the wild cards box.

6.         If necessary, click Move Up, to move the filter to the required position.

7.         Click OK.
